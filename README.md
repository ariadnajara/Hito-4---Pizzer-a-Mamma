# 🍕 Pizzería Mamma Mia

Aplicación web desarrollada con React que consume una API externa para mostrar información de pizzas. Este proyecto fue creado como parte de un desafío de consumo de APIs.

---

## 🚀 Tecnologías utilizadas

* React
* JavaScript 
* CSS
* Node.js
* Fetch API

---

## 📦 Estructura del proyecto

```
REACT/
├── backend_pizzas/        # API REST (backend)
└── pizzeria-mamma-mia/   # Aplicación React (frontend)
```

## 🍕 Funcionalidades

* Mostrar listado de pizzas
* Ver detalle de una pizza
* Render dinámico con consumo de API
* Manejo de imágenes desde backend
* Uso de hooks (`useEffect`, `useState`)

---

## 🧠 Aprendizajes

* Consumo de APIs con Fetch
* Manejo de estados en React
* Uso de `useEffect`
* Configuración de backend con Express
* Servir archivos estáticos (imágenes)
* Manejo de rutas entre frontend y backend

---

## ⚠️ Notas importantes

* Asegúrate de tener ambos servidores corriendo (backend y frontend)
* Las imágenes se sirven desde el backend (`/images`)
* No subir `node_modules` al repositorio
